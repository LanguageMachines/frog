A full class may be N(soort,ev,basis,zijd,stan), so the head is N.

For every value in \textquotesingle{}soort,ev,basis,zijd,stan\textquotesingle{} we lookup the subset in the cgn\+\_\+subsets, and when the constraints on the head are satisfied we return the subset.

For instance\+: \textquotesingle{}soort\textquotesingle{} is found in the subsets to belong to the subset \textquotesingle{}ntype\textquotesingle{} and there are no \textquotesingle{}head\textquotesingle{} constrainst on ntype, so the lookup for \textquotesingle{}soort\textquotesingle{} yields \textquotesingle{}ntype\textquotesingle{}
